Overview

The CNC Pen Plotter is a compact, 3-axis automated machining system designed to replicate digital vector graphics and handwriting onto physical surfaces. While traditional CNC systems are often complex and expensive, this project engineers a cost-effective solution using accessible open-source hardware.

The system bridges the gap between digital design and physical prototyping. By interpreting G-code commands, it coordinates NEMA 17 stepper motors to execute precise pen movements, making it an ideal tool for PCB layout drawing, educational robotics, and automated technical sketching.

Key Features

  • 3-Axis Precision Control: Orchestrates synchronized movement across X, Y, and Z axes for accurate 2D plotting and pen actuation.
  • G-Code Interpretation: Native support for standard G-code instructions generated from vector software like Inkscape.
  • Modular Architecture: Features a fully 3D-printed mechanical frame, allowing for easy customization, repair, and assembly.
  • Automated Z-Axis: Utilizes an MG90S servo motor for precise "pen-up/pen-down" operations to create clean, discontinuous lines.
  • Versatile Tooling: While optimized for pens, the carriage design is adaptable for laser modules or engravers for varied manufacturing tasks.

Architecture & Hardware Stack

The system follows a standard CNC control loop, processing digital signals into mechanical motion.

  • Controller: Arduino Uno R3 running GRBL firmware serves as the central brain, interpreting serial data and calculating motor timing.
  • Drive System: CNC Shield V3 populated with DRV8825 drivers regulates high-current power to the motors, ensuring thermal protection and smooth micro-stepping.
  • Actuators:
    • X & Y Axes: NEMA 17 Stepper Motors provide high-torque positioning for the gantry.
    • Z Axis: MG90S Servo handles the tool-head lift mechanism.
  • Software Interface: Universal G-Code Sender (UGS) streams commands from the host computer via USB.

How It Works

  1. Vector Design: A design (e.g., a PCB trace or logo) is created in software like Inkscape and converted into G-code coordinates.
  2. Command Transmission: The G-code file is loaded into Universal G-Code Sender, which streams data to the Arduino via a COM port.
  3. Execution: The Arduino parses the coordinates (e.g., G01 X10 Y20) and directs the stepper drivers to move the axes by specific step counts.
  4. Plotting: The machine physically draws the path, lifting the pen (Z-axis) when moving between non-connected lines.

Real-World Applications

  • PCB Prototyping: Drawing circuit layouts directly onto copper clad boards for etching.
  • Automated Documentation: Replicating handwritten notes or signatures at scale.
  • STEM Education: Demonstrating principles of mechatronics, coordinate geometry, and digital fabrication.

Sample G-Code Workflow

; Example G-Code executed by the plotter
G21        ; Set units to millimeters
G90        ; Absolute positioning
G01 Z5.0   ; Lift pen (Servo Up)
G00 X0 Y0  ; Move to origin

; Start drawing a line
G00 X10 Y10 ; Move to start point
G01 Z0.0    ; Lower pen (Servo Down)
G01 X50 Y50 ; Draw line to (50, 50)
G01 Z5.0    ; Lift pen
